# pulse

Терминальный дашборд в духе btop: крипта, сеть, погода и время в ncurses-панелях с рамками. Рассчитан на **Linux** и **Termux** (телефон с открытой клавиатурой).

## Быстрый старт

```bash
git clone git@github.com:mrvi0/termux-pulse.git
cd termux-pulse
python3 pulse.py
```

Остановка: **Ctrl+C**. Зависимости из pip не нужны.

## Примеры

### Linux / WSL — полная сетка (широкий терминал)

```bash
# удобно задать размер, например 95×17
printf '\e[8;17;95t' 2>/dev/null || true
python3 pulse.py
```

### Termux — одна панель с клавиатурой (режим **micro**)

Мало строк (портрет, клавиатура открыта) — всё в блоке **PULSE**, без пустых полос:

```bash
pkg install python
cd ~/termux-pulse
python3 pulse.py
```

Примерный вид (~40×12):

```
┌─ PULSE ────────────────────────────────────────┐
│ MSK 14:32  21.05   UTC 11:32  21.05            │
```

### Bash-версия (legacy)

```bash
chmod +x pulse
./pulse --watch 5
# или интервал через переменную
PULSE_INTERVAL=10 ./pulse
```

Нужны: `bash`, `curl`, `jq`, `awk`, `date`, `ping`, `tput`.

### Проверка без полноэкранного TUI

Убедиться, что Python и сеть доступны:

```bash
python3 -c "import curses; print('curses OK')"
curl -sI https://api.coinbase.com/v2/prices/BTC-USDT/spot | head -1
```

## Что на экране

| Панель | Содержимое |
|--------|------------|
| **INFO** | MSK / UTC, погода (СПб), uptime |
| **FEED** | Статус API, сессионные % BTC/ETH |
| **CRYPTO** | BTC, ETH, USDT/RUB (Coinbase spot) |
| **NETWORK** | Внешний IP, пинг Cloudflare / Google |
| **MARKET** | Momentum, волатильность, sparkline |

Данные обновляются каждые **5 секунд**. Перерисовка — при новых данных или изменении размера терминала.

## Адаптивная раскладка

| Режим | Когда | Вид |
|-------|--------|-----|
| **full** | Широкий экран (например 95×17) | Сетка 2×2 + MARKET внизу |
| **compact** | Средний размер | Полосы или 3×2 |
| **minimal** | Очень узко | INFO + CRYPTO + MARKET |
| **micro** | Мало строк (клавиатура Termux) | Одна панель **PULSE** |

## Источники данных

- **Coinbase** — spot: `BTC-USDT`, `ETH-USDT`, `USDT-RUB`
- **wttr.in** — погода
- **api.ipify.org** — внешний IP
- **ping** — `1.1.1.1`, `8.8.8.8`

Нужен исходящий интернет.
